simple_app_config by phuesler

A little helper to define and access the app configuration

Generates a configuration object: * Pass in a YAML file * Pass in an environment as a filter * Use YAML defaults * Use ERB * Returns an OpenStruct so top level attributes can be accessed via method

Install

gem install phuesler-simple_app_config or gem install simple_app_config
